saptamI-vibhakti-dvivachana-vivakShAyAm i.e. when we want to use the word, rAma in the sense of location (saptamI-vibhakti) and in dual (dvivachana), we add the subanta-pratyaya associated with same vibhakti and vachana, which is 'Os', to it.

rAma + Os

The rest of the prakriyA is same as that ShaShThi-vibhakti-dvivachana, rAmayOH
